JavaScript Programmering

String

String er kodningens måde at bruge tekst på. For at opnå dette, skal man blot anvende anførselstegn.

Eksempel på String

const firstName: 'Peter';
const lastName: 'Larsen';
const result: 'My name is ${firstName} ${lastName}';'

Number

Number ligger i ordet, som betyder tal.

Eksempel på Number

const lommeregner: 5 + 5;
console.log(lommeregner); // 10

Booleans

Booleans kan man bruge, for at se om noget er true eller false. Man benytter booleans i 'if statements' og for 'loops'.

Eksempel på Boolean

const age = 15; if (age >= 18) {
console.log('You can drive');
console.log('Congratulation homie!');
} else if (age >= 17) {
console.log('Almost there brother!');
} else {
console.log('You can not drive'); }

Det vil altså sige, at man her undersøger om personen er 18 og derved gammel nok til at køre bil. Hvis dette er tilfældet (true), så gives beskeden "You can drive". Er det ikke tilfældet og personen er 17 (næsten gammel nok), så gives beskeden "Almost there brother.". Er personen derimod under 17 år, gives beskeden "You can not drive.". Og på den måde bruges booleans.

Function

Koden funtion er en kodelinje som udfører en meget specifik opgave.

Eksempel på Function

function greet () {
console.log('Hello stranger');
}
Ønsker man at fremkalde denne funktion, kan man blot skrive:
greet ();
og så vil den hilsen man har lagt ind, komme frem i konsollen.

Ofte vil man dog støde på en function som er sat sammen med en parameter. Man kan fx have en variable som hedder navn, hvor man beder brugeren indtaste sit navn og når der bliver trykket "enter", så kommer der en besked op.

Eksempel på function+parameter

function greet(navn) {
console.log("Hello" + navn");
}

let name = prompt ("Enter your navn");
greet(name);

Variabler

Den bedste måde at beskrive en variable på, er ved at forestille sig at en variable er en kontainer. Denne kontainer indeholder data. Lad mig give et eksempel.

Eksempel på Variabler

let num1 = 5;
let num2 = 10;

const num1 = 5;
const num2 = 10;
Der er to måder at skrive en variable på, nemlig med enten "let" eller "const". Forskellen er egentligt super simpel. Hvis man regner med at værdien af ens variable ændres løbene, er det nødvendigt at bruge let. Her er det nemlig muligt at ændre værdien, med en simpel kodelinje såsom:
num1 = 6.
Nu har vi ændret værdien num1 fra 5 til 6. Bruger man derimod const (konstant), kan værdien ikke ændres.
